# 涛笺桌面客户端

> 墨染流年，笺藏春秋

一款文艺风格的中文便签/笔记管理工具。

## 一键启动（推荐）

双击 **`start.vbs`** 即可启动！

- 无命令行黑窗口
- 启动时显示古风加载画面（约 1.5 秒）
- 主窗口准备好后自动进入

## 打包为独立安装程序（可选）

如果你希望生成真正的 `涛笺 Setup.exe`：

1. 安装 Node.js: https://nodejs.org/
2. 双击 `build.bat`
3. 打包完成后在 `dist/` 目录找到：
   - `涛笺 Setup.exe` — 安装程序
   - `dist\win-unpacked\涛笺.exe` — 免安装版

## 功能

| 功能 | 说明 |
|------|------|
| 四大模块 | 工作、生活、纪念、密码袋 |
| 完整 CRUD | 新建、编辑、删除、搜索 |
| 数据导入/导出 | JSON 格式备份与恢复 |
| 自动保存草稿 | 编辑时每 5 秒自动保存，关闭弹窗自动清除 |
| 按时间排序 | 最新编辑的条目置顶 |
| 窗口状态记忆 | 重启后恢复上次位置和大小 |
| 本地数据存储 | 数据保存在本地 JSON，安全可靠 |

## 启动方式对比

| 文件 | 用途 | 黑窗口 |
|------|------|--------|
| `start.vbs` | 一键启动（推荐） | 无 |
| `start.bat` | 备用启动（带提示） | 有 |
| `build.bat` | 打包为安装程序 | 有（仅打包时） |

## 数据位置

所有数据保存在 `data/` 目录：
- `work.json` — 工作
- `life.json` — 生活
- `memorial.json` — 纪念
- `password.json` — 密码袋

## 项目结构

```
涛笺/
├── start.vbs          ← 一键启动（无黑窗口）
├── start.bat          ← 备用启动
├── build.bat          ← 打包脚本
├── main.js            ← 主进程（窗口管理、数据持久化）
├── preload.js         ← 安全桥接（暴露 API 给前端）
├── package.json       ← 项目配置
├── renderer/
│   ├── index.html     ← 主界面（登录、主页、子页面）
│   └── splash.html    ← 启动加载画面
├── assets/            ← 图片和图标
├── data/              ← 本地数据（JSON）
└── electron/          ← Electron 运行时
```
